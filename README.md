A napari plugin for loading, visualizing, and editing centerlines generated by **WormTracer**, frame-by-frame, and saving the refined results. This tool is designed for workflows where manual refinement of centerlines or skeleton paths is needed across time-lapse or multi-frame data.

## Data Formats

* **Mask/Image File**: Typically a binary or grayscale mask (.tif, .avi, ...etc).
* **Centerline File**: A `.csv` file containing the centerline coordinates exported from **WormTracer**. (ex:  `sample_1_x.csv` and `sample_1_y.csv` in the same folder) or a `.hdf` file containing two datasets, namly "x" and "y", respectively.

## Features

This plugin adds a small control panel with convenient buttons:

| Button              | Description                                                                            |
| ------------------- | -------------------------------------------------------------------------------------- |
| **Load Image**      | Load an image or image stack into napari.                                              |
| **Load Centerline** | Load an existing centerline or set of centerlines.                                     |
| **Prev (-1)**       | Move to the previous frame.                                                            |
| **Next (+1)**       | Move to the next frame.                                                                |
| **Clear**           | Clear the current centerline modification                                              |
| **Reload**          | Discard all modifications and reload the centerline data from the original source.     |
| **Flip**            | Flip the head and tail orientation.                                                    |
| **Register**        | Apply registration or alignment adjustment to the current frame.                       |
| **Undo**            | Undo the last edit.                                                                    |
| **Save All**        | Save all edited centerlines back to file.                                              |

## Usage

1. Open napari.
2. Open the plugin from the napari plugin menu.
3. Use **Load Image** to open your image data.
4. Use **Load Centerline** to load centerline coordinates.
5. Navigate frames with **Prev** / **Next**, adjust centerlines as needed.
6. Use **Save All** once all frames are reviewed and corrected.

* napari

## Installation

You can install `napari-wormtracer` via [pip]:

```
pip install napari-wormtracer
```

If napari is not already installed, you can install `napari-wormtracer` with napari and Qt via:

```
pip install "napari-wormtracer[all]"
```
